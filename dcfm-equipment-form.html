DCFM Equipment Form

Data Center Facility Management form for new and unrecorded equipments.

Developer Notes:
The Equipment ID will be automatically generated based on the selected DC Location and Equipment Type. The format will be: [DC Location]-[Equipment Type]-[DB GENERATED SEQUENCE]. For example, if the DC Location is "KVDC" and the Equipment Type is "UPS", the generated Equipment ID might look like "KVDC-UPS-00001". The sequence number will be generated by the database for DCFM Equipments to ensure uniqueness preferable 5 digits starts with 00001. This auto-generation logic will be implemented in the backend, and the frontend will simply display the generated ID in the disabled input field. The auto-generated ID is processed in dcclarity-global-var.js while the data pointer is located in the dcfm-equipment-form.html script area file.

The Equipment Type & DC Location dropdowns will be populated with predefined values located in the dcclarity-global-var.js file. The frontend will fetch these values and populate the dropdowns accordingly when the form loads. This ensures that any update to the Equipment Types or DC Locations can be easily managed by updating the global variable file without needing to change the frontend code.
