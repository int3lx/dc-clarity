Developer Notes

This is not an auto-load list. Request to load list only be done when the "Load List" button clicked. The "DC Location" & "Equipment Type" work as filter to speed-up list loading. Leaving both unchanged will allow full list to be loaded from the database.

The list is loaded from the database as per table header placement with added extra column "Actions". View button will be created along with the DB data load and function as a link to dcfm-equipment-details.html to a spesific $dcfm_equipment_id

The DC Location & Equipment Type dropdown work as a filter Input in the table header act as a filter to related column.

The list table is set to auto overflow.

Database refer to dcfm-equipment-db.html
